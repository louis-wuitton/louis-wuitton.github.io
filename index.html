ViaSat

June 2017 - September 2017, San Jose, CA

During my summer internship at ViaSat, I built a network controller dashboard for ViaSat's overlay virtual network team. The network controller dashboard is a visualization tool to display SDN overlay network across multiple data centers. In a SDN network, the system that makes decisions about where traffic is sent (SDN Controller) is decoupled from the underlying systems that forward traffic, and usually these underlying systems are virtual machines, which could communicate to each other through protocols like OpenFlow. Network controller dashboard could show you all the virtual machines within SDN, and how are they communicated to each other through network bridges.
